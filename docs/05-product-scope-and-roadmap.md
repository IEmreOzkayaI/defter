# Product scope + roadmap (Vaha)

## Urun tanimi (MVP)

Vaha; hamam / sauna / spa isletmelerinde:

- Oturum ac
- Hizmet kalemi ekle
- Sureyi izle
- Tahsilatla kapat

## Kapsam ici (MVP)

- Kabin / anahtar preset listesi
- Hizmet katalogu (hamam, sauna, masaj, kafeterya, paket)
- Gun sonu ozeti (demo)
- Isteg bagli POS gonderimi (demo)

## Kapsam disi ( simdilik )

- Tam CRM / uyelik yonetimi
- Muhasebe / e-fatura
- Stok yonetimi

## Yol haritasi

1. Online rezervasyon + doluluk
2. Personel ve vardiya
3. Gercek POS entegrasyonlari + odeme durumu
4. Sube / tenant modeli (coklu sube)

## SEO hedef kelimeler (TR)

- Isletme yazilimi: `hamam spa otomasyon`, `sauna yazilimi`, `spa yonetim programi`
- Rezervasyon niyeti: `hamam rezervasyon`, `sauna rezervasyon`, `spa randevu yazilimi`
